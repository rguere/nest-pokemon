## Ejecutar en desarrollo

1._ Clonar el repositorio

2._ Ejecutar

```

yarn install

```
3._ Pre requisito: tener instalado Nest CLI instalado

```

npi i -g @nestjs/cli

```
4._ Levanatar la base de datos

```

docker-compose up -d

```

## Stack usado
* MongoDB
* NestJS
* TypeScript
